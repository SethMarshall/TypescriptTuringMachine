Typescript Turing Machine

            This application implements a Turing Machine which computes the set of binary strings beginning with any number of "1" followed by twice as many "0".
For example, it will accept 110000 and 111000000, but not 1100000000, and obviously not 1001001 or 0011. Note that it decides whether an input string
is valid, not by couting or comparing to any other data, but purely through "raw" Turing machine read/write operations.

Enter a binary string (up to 21 digits) to try it out!
